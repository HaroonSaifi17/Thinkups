A simple angular Website for a Client

### Setup:

1. **Clone the Repository:**
   ```
   git clone https://github.com/HaroonSaifi17/ThinkUps.git
   ```

2. **Install Dependencies:**
   ```
   npm install
   ```

4. **Run the Server:**
   ```
   npm run test 
   ```

5. **Run the Client:**
   ```
   ng serve
   ```
